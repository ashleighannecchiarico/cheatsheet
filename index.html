HTML - HyperText Markup Language

HTML Basics to Include:

Tag: Description: Element Type:
<!DOCTYPE html> Notates the type of document for the browser to expect. Always required as the first line of code. Open
<html> Known as the root element. It should wrap all content on the page. It is the second line of code. Closed
<head> Acts as a container for content you want to include but not show to your page's viewers. CSS stylesheet goes here. Closed
<title> Sets the title of the page in the tab of the browser. Used as the title of a favorite/bookmark. Closed
<body> Showcases all of the content you want your viewer's to see. Closed

Code Example:

<!DOCTYPE html>
<html lang="en-US">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>My test page</title>
</head>
<body>
  <img src="images/firefox-icon.png" alt="My test image">
</body>
</html>

Tables:

Tag: Description:
<table> Defines a table
<caption> Defines a table caption
<th> Defines a header cell in a table
<tr> Defines a row in a table
<td> Defines a cell in a table
<thead> Groups the header content in a table
<tbody> Groups the body content in a table
<tfoot> Groups the footer content in a table
<col> Specifies column properties for each column within a <colgroup> element
<colgroup> Specifies a group of one or more columns in a table for formatting

Code Example:

    <table>
  <tr>
    <th>Company</th>
    <th>Contact</th>
    <th>Country</th>
  </tr>

CSS - Cascading Style Sheets

CSS Syntax


  h1 { 
    color:blue;
    font-size:12px;
  }

The selector (h1) points to the HTML element you want to style.
The declaration block contains one or more declarations separated by semicolons.
Each declaration includes a CSS property name and a value, separated by a colon.
Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

How to Add CSS

Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section.

Code Example:


<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="mystyle.css">
  </head>
  <body>
  </body>
</html>
